DRIVE SETUP GUIDE

  1. Go to console.cloud.google.com and create a new project (or pick an existing one).
  2. In the left menu, open APIs & Services → Library, search Google Drive API, and click Enable.
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID.
  4. Choose Web application. Under "Authorized JavaScript origins" add the exact URL you're hosting OMAR on (e.g. https://yourname.github.io).
  5. Click Create. Copy the Client ID (ends in .apps.googleusercontent.com).
  6. Paste it into OMAR's Settings → Google Drive → OAuth Client ID, save, then tap Connect Drive.

You only do this once. The Client ID isn't a secret password — it just tells Google which app is asking — but only ever add origins you actually control.
